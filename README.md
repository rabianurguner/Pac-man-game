# Pac-man Oyunu
Bu proje, klasik Pacman oyununun JavaScript, HTML ve CSS kullanılarak geliştirilmiş basit bir versiyonudur. Oyun, web tarayıcısı üzerinde çalışır ve temel oyun mekaniklerini barındırır:

# Özellikler
•Tamamen HTML, CSS ve JavaScript ile geliştirilmiştir.
•Responsive ve modern bir arayüz.
•Temel Pacman yapay zekasına sahip hareketli hayaletler.
•Oyun alanı grid yapısı ile oluşturulmuştur.
•Puan sistemi ve oyun sonu ekranı.

# Kullanılan Teknolojiler
•HTML5
•CSS3
•JavaScript (Vanilla JS)

# Nasıl Çalışır
Sunucuya gerek kalmadan `index.html` dosyasını herhangi bir tarayıcıdan çalıştırmanız yeterli olacaktır.
