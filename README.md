# Pac-man-game
Tüm zamanların en ikonik arcade oyunu şimdi tarayıcınızda! Pac-Man olarak labirentte ilerleyin, tüm puanları (dots) yiyin ve hayaletlerden kaçın. Enerji Küresi'ni (Power Pellet) kapın ve avcıdan av konumuna geçin! Basit, bağımlılık yapıcı ve nostaljik bir arcade keyfi sizi bekliyor. Rekorunuzu kırmaya hazır mısınız?
